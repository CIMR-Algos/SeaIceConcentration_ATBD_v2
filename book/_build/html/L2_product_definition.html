Level-2 product definition#

The CIMR Level-2 SIC, SIC1H, and SIED products are on instrument grid, thus with a (Nscanl,Nscanp) structure. We describe below the main variables for each type of products. Each product file will in addition have geolocation information (latitude and longiture) and attributes, that are not described here at this stage.

At this stage, it is foreseen that the SIC3H product files can have several SIC estimates, from different channel combinations. Each SIC will have its own group, similar to that in Table 1.

Sea Ice Concentration 1H (SIC1H)#

The SIC1H product files will have variables similar to those in Table 1, possibly less of them (e.g. no uncertainty variable) if these have to be skipped for timeliness requirements. By the same token, we expect only one SIC algorithm to run within the NRT1H chain and thus only one SIC group in the data file. Finally, depending on the configuration of the NRT1H chain, selected brightness temperature channels might also be shipped in the SIC1H product file, to help downstream applications within sea-ice navigation safety (e.g. regional sea-ice drift and type monitoring). Candidate microwave channels are Ku (H- and V-pol) and Ka (H- and V-pol).
